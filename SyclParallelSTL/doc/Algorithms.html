Algorithms

This document describes the implementation progress of the algorithms included in the Parallel STL standard. In addition, it includes notes on the iterators that our implementation can recieve/return, along with other notes on the implementations.

Note: implemented is "as of Dundee" release, not counting in-progress algorithm development.

If only one iterator specified, the current minimum/maximum iterator is also the ideal one.
